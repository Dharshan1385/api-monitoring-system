# API Monitoring System 🚀

A real-time API monitoring system built using Flask to track API health, response time, and status.

---

## Features
- API health check (HEALTHY / DEGRADED / DOWN)
- Response time tracking
- Real-time dashboard
- Modular Python design

---

## Tech Stack
- Python
- Flask
- Requests

---

## How to Run
pip install -r requirements.txt  
python app.py  

Open: http://127.0.0.1:5000
