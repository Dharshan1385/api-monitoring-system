# API Monitoring System

Live Demo: https://api-monitoring-system-v191.onrender.com

GitHub Repository: https://github.com/Dharshan1385/api-monitoring-system

## Project Overview

API Monitoring System is a Python Flask-based web application that continuously monitors API health, response times, uptime percentage, and service availability. The application provides a real-time dashboard with charts, status tracking, and alert logging to help identify service outages and performance issues.

## Features

* Real-time API health monitoring
* Response time tracking and visualization
* Uptime percentage calculation
* Healthy, Degraded, and Down status classification
* Interactive dashboard with charts
* Alert history tracking
* Twilio SMS notifications for outages and recovery events
* Cloud deployment using Render
* Publicly accessible web dashboard
